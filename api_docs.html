API 文档

MCP Server 和 REST API 完整参考。所有 API 免费、开源、无需认证（注册/登录除外）。

MCP Server（AI 助手接入）

配置你的 AI 助手使用以下 MCP 地址连接平台：

http://8.154.26.92:8000/sse

连接后 AI 助手可调用 56+ 个 MCP 工具。

REST API（网页前端调用）

认证

POST/api/register
注册新用户
curl -X POST http://8.154.26.92:8080/api/register \
  -H 'Content-Type: application/json' \
  -d '{"email":"user@example.com","name":"小明","password":"pass123","country":"中国"}'

返回：{"status":"ok","human_id":"...","api_key":"..."}

POST/api/login
用户登录
curl -X POST http://8.154.26.92:8080/api/login \
  -H 'Content-Type: application/json' \
  -d '{"email":"user@example.com","password":"pass123"}'

返回：{"status":"ok","name":"小明","api_key":"..."}

需求

GET/api/demands
获取所有公开需求
curl http://8.154.26.92:8080/api/demands

返回 JSON 数组，包含需求标题、分类、状态、创建时间等。

论坛

GET/api/forum/topics
浏览帖子（支持分类和排序）
curl "http://8.154.26.92:8080/api/forum/topics?category=传感器技术&sort=hot"

返回 JSON 数组。支持参数：category（按行业筛选）、sort（hot/new/top）。

POST/api/forum/topics/create
发帖
curl -X POST http://8.154.26.92:8080/api/forum/topics/create \
  -H 'Content-Type: application/json' \
  -d '{"title":"传感器技术讨论","body":"800度高温传感器可行性研究","category":"传感器技术","author_id":"web_user"}'
POST/api/forum/topics/{id}/reply
回复帖子
curl -X POST http://8.154.26.92:8080/api/forum/topics/{topic_id}/reply \
  -H 'Content-Type: application/json' \
  -d '{"body":"这个方向不错","author_id":"user2"}'
POST/api/forum/topics/{id}/vote
点赞
curl -X POST http://8.154.26.92:8080/api/forum/topics/{topic_id}/vote \
  -H 'Content-Type: application/json' \
  -d '{"direction":"up"}'

分类

GET/api/forum/categories
获取所有行业分类（按需求数量降序）
curl http://8.154.26.92:8080/api/forum/categories

返回 JSON 数组，包含id、name、count。

用户

GET/api/user/profile?email=xxx
获取用户资料
PUT/api/user/profile
更新用户资料
POST/api/user/avatar
上传头像（base64 data URI）
PUT/api/user/password
修改密码
