## 五、API 接口

| 端点 | 方法 | 说明 |
|:-----|:----|:----|
| `/api/suppliers` | GET | 供应商列表（page/per_page/keyword） |
| `/api/demands` | GET | 需求列表（page/per_page/keyword/category） |
| `/api/forum/topics` | GET | 论坛主题列表 |
| `/api/forum/categories` | GET | 论坛分类 |
| `/api/forum/topics/{id}` | GET | 论坛主题详情 |
| `/api/forum/topics` | POST | 发布主题 |
| `/api/forum/reply` | POST | 回复主题 |
| `/api/forum/vote` | POST | 投票 |
| `/api/auto-demand` | POST | 爬虫需求入库 |
| `/api/auto-supplier` | POST | 爬虫供应商入库 |

## 六、部署步骤

```bash
# 部署代码
ssh -p 2222 root@demand-chain.duckdns.org
docker exec -i dc-web bash -c 'cat > /app/<文件>' < 本地文件
docker restart dc-web

# SQL操作
docker exec dc-db psql -U dc -d demand_chain -c "SQL语句"

# 查看日志
docker logs dc-web --tail 50
```
