## 如果发现网站有问题，回档方案

### 方案A：还原到原始镜像

```bash
# 停web容器
docker stop dc-web && docker rm dc-web

# 用original镜像起
docker run -d --name dc-web --restart unless-stopped \
  --network dc_default \
  -p 8080:80 \
  demand-chain:original \
  python -m src.web_server

# 然后重新部署热修复文件
docker cp /tmp/suppliers.html dc-web:/app/suppliers.html
# ... 其他文件
```

### 方案B：用 docker-compose 重建（slim镜像，当前状态）

```bash
cd /opt/demand-chain
docker compose -f docker-compose.prod.yml -p dc up -d web
docker cp /tmp/*.html dc-web:/app/
```
